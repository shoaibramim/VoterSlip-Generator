# VoterSlip Generator - Bengali OCR Backend

This Space provides OCR extraction for Bengali voter information from PDF files.

## Features
- Bengali text recognition using Tesseract OCR
- Extracts voter names, numbers, father/mother names
- Processes multiple pages efficiently
- Returns structured JSON data

## Usage
Upload a PDF file containing Bengali voter lists to extract structured voter data.

## API Access
This Space can be used as an API endpoint for your frontend application.
